## クイックスタート

### 使ってみたい方

```bash
# airc を実行してデフォルト設定をダウンロード
npx @maru3460/airc
```

### 開発したい方

```bash
# リポジトリをクローン
git clone https://github.com/maru3460/airc.git
cd airc

# 依存関係をインストール
npm install

# ビルド
npm run build

# ローカルで実行
node bin/cli.js
```
